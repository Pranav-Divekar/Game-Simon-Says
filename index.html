How to Play Simon Game

1. Press any key to start the game.

2. Watch the sequence of colors.

3. Repeat the sequence by clicking the colored buttons in the same order.

4. The game gets harder as the sequence gets longer.

5. If you make a mistake, the game is over and you can start again.
